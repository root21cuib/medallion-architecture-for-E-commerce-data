# medallion-architecture-for-E-commerce-data

This  project had objective to ingest data from transactions that took place on the olist commerce website. In addition to that, merge the data with information related to the platform's supply and shipment processed and with that build a curated dataset on which a semantic model is built to be perform to perform Business Intelligence analysis  using PowerBI tools and sales predictions using python frameworks.

The outcome fo this project is to provide stakeholder with information concerning the health, growth and KPIs of the busienss to promote scale and evolution
